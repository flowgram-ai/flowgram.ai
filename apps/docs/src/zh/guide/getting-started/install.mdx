# 安装

import { PackageManagerTabs } from '@theme';

## 通过 npx 安装

<PackageManagerTabs command={{
  npm: "npx @flowgram.ai/create-app@latest"
}} />

```shell
# 选择 demo
- fixed-layout # 固定布局最佳实践
- free-layout # 自由布局最佳实践
- fixed-layout-simple # 固定布局基础用法
- free-layout-simple # 自由布局基础用法

```
